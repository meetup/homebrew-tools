# Homebrew tools

Formule for installing tools on osx

```sh
$ brew tap meetup/tools
```

## git-linecat

```sh
# (optional) brew update
$ HOMEBREW_NO_INSTALL_CLEANUP=1 brew install git-linecat
```

## rebazel

```sh
# (optional) brew update
$ HOMEBREW_NO_INSTALL_CLEANUP=1 brew install rebazel
```

## stack-tail

```sh
# (optional) brew update
```
